Samba Configuration

* Target Configuration

Follows these steps to configure Samba on target machine

1. Connect a Pendrive/HardDisk to the target

2.Modify the Sharepath (/mnt/usb/XXXXXX/Disc-X) and apply

* Host Configuration

Follows these steps to configure Samba on host machine

1.Go to HOST Machine In a windows m/c. go to Run --> Execute \\192.168.1.1\

2.Give the credentials as username: ugw password: ugw

3.Now a separate window is opened which shows the list of all the files/dir which is exactly same as the mounted partition of USB 4. Now perform transfer of files from USB to local machine and vice-versa

If a new user want to be created and access samba. Create a new user from Advanced--> User Mangement --> Enable User --> Give Samba Access and apply,Follow the same steps as above
